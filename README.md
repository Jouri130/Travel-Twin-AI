# Travel Twin AI 🧭

تطبيق ذكي يعتمد على نماذج "عِلم" لبناء توأم رقمي للسائح، يقوم بتخطيط رحلات سياحية مخصصة داخل المملكة العربية السعودية بناءً على الاهتمامات والظروف اللحظية.

## ✨ المميزات الرئيسية
- **التوأم الرقمي (User Twin):** تحليل سمات المستخدم وتفضيلاته لحظياً.
- **التخطيط الذكي:** استخدام نموذج `Nuha-2.0` لتوليد جداول سياحية مرنة.
- **الذكاء السياقي:** مراعاة حالة الطقس والازدحام في التوصيات.
- **التفاعل الصوتي:** تحويل التوصيات إلى صوت طبيعي بلهجة سعودية عبر `Elm-TTS`.

##البنية التقنية للمشروع:
تعتمد البنية التقنية للمشروع على فصل واجهة المستخدم عن منطق الذكاء الاصطناعي. نستخدم ملف user_data.json كنموذج هيكلي (Data Model) يوضح آلية تنظيم بيانات المستخدم وتفضيلاته. حالياً، يتم التعامل مع البيانات ديناميكياً داخل التطبيق لضمان التكامل مع نموذج nuha-2.0 وتفعيل خاصية التكيف اللحظي مع الازدحام والطقس.

## 🛠️ التقنيات المستخدمة
- **Language Models:** Nuha-2.0 (Elm Models).
- **Speech Synthesis:** Elm-TTS.
- **Framework:** Streamlit.
- **Logic:** Python, JSON Structuring.
